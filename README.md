# Admin Panel For blog

## Dependencies
1. Vue.js
2. Apollo.js
3. Electron-vue
4. npm


## Getting Started


``` diff
# clone the project
  git clone https://github.com/Ayoub-Bouarbi/blog-admin.git

# enter the project directory
  cd blog-admin

# install dependency
  npm install

# start
  npm run start

# to run electron app
  npm run electron:serve

```
